How to demo

  1. Set your display name in the top-right input (optional).
  2. Type a message and press Enter or click Send.
  3. The app shows your message and a simulated reply (bot) after a short delay.
  4. Use Clear to clear the chat area.

Features

  • Responsive layout
  • Message timestamps
  • Auto-scroll and keyboard support
  • Light-accessible color contrast
